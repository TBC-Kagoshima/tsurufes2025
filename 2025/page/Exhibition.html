#　文化祭開催に当たって

前期生徒会文化局長 神野柚葉
　ついに文化祭の季節がやってきました! 伝統ある鶴丸高校の文化祭運営に文化局長として携われることをうれしく思います。 今年の文化祭では、そんな伝統に風を吹き込む新たな取り組みもいくつか導入しました。 「鶴志燃翔」のスローガンのもとで、生徒のみなさんの燃える志を感じられることを期待しています。 最後に、今回の文化祭運営に協力してくださった文化祭実行委員はじめ生徒の皆さん、先生方、生徒会の皆さん、そしてともに苦楽を分かち合った文化副局長ならびに文化局員、放送部、先輩方に心より感謝申し上げます。 それでは鶴丸高校文化祭開幕です! みなさまお楽しみください!

#　ダンス部

　みなさんこんにちは!ダンス部です。 私たちは2年生10名、1年生10名で、リズムダンスや創作ダンスの練習に励んでいます。 今年も文化祭のオープニングを飾れ、嬉しい気持ちでいっぱいです。 一緒に楽しみましょう!

#　演劇部

　生徒館2階の隅っこで静かなようで賑やかに緩く活動している演劇部9名が本日お届けするのは、「放課後の宴会」です。 どこかの学校のどこかの教室で起こる奇妙で笑えるお話を、ぜひお楽しみ下さい。

#　音楽部

　9月に先輩方が引退され、新体制となった鶴音の最初のステージは『あなたに贈る希望のメドレー』です! 一人ひとりのソロパートに大注目して、素敵な音色で新生鶴音を勢いづける私たちのステージをお楽しみください。

#　書道部

　私たちはこれまで全員で意見を出し合い、文字の綺麗さだけでなく、曲や構成など様々なこだわりをもって準備をしてきました。 一筆一筆に心を込め、見てくださる皆さんの心に届くような力強い書を披露します。

#　ESS部

　Have you ever heard debate? 私たち ESS 部は放課後、英語で熱き戦いを繰り広げています。 立論、反論、再構築。 白熱した応酬。 普段目にすることのない、私たちの勇姿を是非ご覧ください!

#　放送部

　今年の放送部は例年とは異なり、ニュース風に運動部を取材! 部員の声と共に部活動の魅力を紹介します。 また、腕相撲で部活動最強を懸けた熱戦の様子もお届けします!

#　吹奏楽部

　新体制になって 2 ヶ月。 1・2年で演奏する初めてのステージです。 みなさんもよく知る人気バンドのあの曲やジブリ・吹奏楽の名曲など盛り上がること間違いなしの曲ばかりです。 皆で歌って盛り上がって楽しみましょう!

#　The Brights

　Hello. The Brights です。 私たちはほとんどが初心者でとても苦戦しましたが、みんなで協力して少しずつできるようになってきました。 一緒に盛り上がっていきましょう!
