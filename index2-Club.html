部活動・同好会

#　科学部 / 1F 化学実験室

　かのアルキメデスは言った。 「エウレカ!!︎エウレカ!!︎」と。 我々の日常は発見で満ち満ちている。 そのありふれた発見に意味を見出し、尊ぶ。 それが化学部である。 そんな化学部は今回の文化祭でとある壮大な実験を行う。 我々が行う《絶対零度圏域展開実験》を刮目せよ。

#　茶道部 / 草寿庵・記念庭園

　私たち茶道部は「秋鶴苑」を全体のテーマとし、草寿庵では「一日千秋」、記念庭園では「客来一味」をテーマにお茶会を開催します。 心を込めて点てたお抹茶と季節のお菓子でお客様をおもてなしいたします。 日常を離れて、静かなひととき、心落ち着く時間を過ごしませんか? 部員一同お待ちしております。

#　書道部・吹奏楽部・音楽部 / 中庭

　青空の下、吹奏楽と合唱の鮮やかなハーモニーにのせて、書道部の躍動感溢れる筆使いをご披露します。 1年に 1 度しかない、貴重な 3 部合同パフォーマンスをお楽しみ下さい!

#　生物部 / 1F 生物講義室

　飼育している生き物、標本、写真の展示に加え、今年からスタートした研究内容の紹介やツルグレン装置による土壌微生物の観察など、初めての活動にも取り組みました。 また、60 年ぶりに鶴丸高校生物部誌「あこう」が復活します。 記念品も配布するので、ぜひ見に来てください。

#　地学部 / 1F 地学室

　地学部では、文化祭限定の「地学クイズチャレンジ」を開催中! 地球と宇宙にまつわる様々な問題に挑戦してみませんか? 地学室にきてくれた皆さんには全 12 種類のオリジナル星座カードをプレゼント! 地学の知識がなくても楽しめる内容です。 地球の不思議、宇宙のロマンを、ぜひ体験しに来てください!

#　写真部 / 1F 生物実験室

　写真部の部員が活動し始めてから撮影した写真の展示をしています。 学校行事から風景写真、旅先の写真まで幅広いジャンルの写真があります。 気に入られた写真は後日、さし上げることも可能です。 展示会場が人気のない場所ですが是非、ご来場ください。

#　百人一首部 / 文化館 1F 百人一首部室

　私たち百人一首部は、百人一首の体験コーナーや札のキーホルダーなど、百人一首を知らない人でも楽しめる企画をたくさん行います。 今「ちはやふる−めぐり−」でも話題になっている百人一首。 “畳の上の格闘技“と称され、0.1 秒を争う競技かるたの世界を体験してみませんか。 部員一同お待ちしています。

#　山岳部 / 2F 山岳部室

　山岳部にとって初の試みとなる文化祭展示ですが、我々にとってこれは大チャンス! 全くと言っていいほど見てもらう場のない私達の活躍を今日はしっかりお見せします。 様々な展示を部員一同協力して用意しました。 部室を一生懸命片付けてお待ちしておりますのでぜひ足を運んでみてください。

#　クイズ研究会 / 3F 20A 教室

　本校クイズ研究会は今年誕生しました。 展示では来場者の皆さんに楽しんでいただけるクイズ大会やオリジナル問題を準備しました。 文化祭ならではの雰囲気の中で、クイズの面白さをぜひ見つけにいらしてください。 友達と体験したり実力を試したり、クイズの世界へとびこんでみませんか。

#　数学研究部 / 3F 20B 教室

　ただの数学には興味ありません。 この中に宇宙人、未来人、異世界人、数学者がいたら、あたしのところ(数研)に来なさい。 そうでない人も来なさい。 数研は作問や部誌、数学に関係深い展示をするわ。 きっと楽しめるはず。 そのことは来て証明しない。 興味がある人はぜひ訪れてみること。 以上。

#　文芸部 / 2F 階階段前

　こんにちは、文芸部です。 今年の文化祭の部誌は「七不思議」がテーマとなっています。 残暑が続きますが、この一冊で肝を冷やしてはいかがでしょうか? 部員一同、皆様が我々の作品を手に取っていただくことを、心よりお待ちしています。

#　美術部 / 4F 美術室

　美術部は今年の文化祭で「モノクロ」と「カラフル」な世界をテーマに生徒の作品の展示や部誌の配布を行います。 普段美術に触れる機会が少ない人にも楽しんでもらえるような内容になっています。 どの作品にも力を入れており、内容も充実しているので、気軽に足を運んでもらえるとうれしいです。
